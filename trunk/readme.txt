Introduction
------------
NIntegrate is a .NET SOA Integration Framework on the top of WCF.


Goal
------------
The goal of NIntegrate is to implement a rapid development framework for 
.NET developers to apply SOA design patterns easier.


Dependencies
------------
.NET Framework 3.5

------------
